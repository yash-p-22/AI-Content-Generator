

## Project Title

AI Website Content Generator


# Project Goal

Build a **production-quality full-stack web application using Next.js** that generates **structured website content** (Hero section, Services section, FAQ section) from business information using **LLM generation**.

The application must use:

* **Next.js App Router**
* **Server API route for LLM calls**
* **Dexie.js for IndexedDB persistence**
* Client-side **generation rate limiting**
* **JSON schema based structured outputs**
* Export capability
* A **premium portfolio-style UI inspired by the reference image**

The system must work **inside Replit's built-in AI runtime first**, without requiring external API keys unless deployed outside Replit.

# Required JSON Output Schema

The LLM must strictly return this JSON format:

```
{
  "hero": {
    "headline": "",
    "subheadline": "",
    "cta_primary": "",
    "cta_secondary": ""
  },
  "services": [
    {
      "title": "",
      "description": "",
      "icon": ""
    }
  ],
  "faq": [
    {
      "question": "",
      "answer": ""
    }
  ]
}
```

Rules:

* 3 to 5 services
* 5 FAQs
* Tone must match user selection
* Output must be pure JSON
* No markdown
* No explanations

---

# Client Rate Limiting

Users can generate **100 times per 24 hours**

Logic:

1 Query Dexie
2 Filter timestamps within last 24 hours
3 If >= 100 → block generation
4 Show message

Example message:

"You have reached today's generation limit. Try again tomorrow."
